ES6新特性

变量声明

  1. var 变量可以重复声明,且拥有全局作用域

    • 重复声明

      var name ="张三"; 
var name ="李四";
console.log(name);
//>>>输出"李四"
//重复声明即为赋值

    • 全局作用域

      {
    var name="张三"
}
console.log(name);
//>>>输出"张三"

    • 变量提升

      console.log(name);
var name="张三";
// 可以在声明前进行调用
//>>>输出undefined

  2. let 声明的变量为块级作用域, 不影响变量的作用域链,即在当前作用域声明的变量,可在子作用域内使用,而无法在父级作用域中使用.
    变量无法重复声明.应尽量let声明变量

    • 变量不能重复声明

      let name ="张三"; 
let name ="李四";
//错误

    • 块级作用域

      if (true) {
    let name="张三"
}
console.log(name);
//错误

    • 无变量提升

      console.log(name);
let name="张三";
//报错

  3. const 声明一个常量,声明后无法修改,一边常量使用全大写,方便理解

    • 值不能修改

      const NAME = "张三";
console.log("张三");
NAME = "李四";
//错误

    • 必须赋初始值

      const NAME;
//错误
const NAME = "张三";
//正确

    • 块级作用域

      if (true) {
    const NAME="张三"
}
console.log(NAME);
//错误

    • 对数组和对象的修改,不算为对常量的修改

      const NAME = ["张三", "李四", "王五"];
NAME.push("赵六");
console.log(NAME);
//正确

解构赋值

  1. 数组的解构赋值

    const NAME = ["张三", "李四", "王五", "赵六"];
let [n1, n2, n3, n4] = NAME;
console.log(n1, n2, n3, n4);
// >>>张三 李四 王五 赵六

  2. 对象的解构

    const person = {
name: "张三",
age: 18,
say: function () {
    console.log("昨夜星辰昨夜风,画楼西畔桂堂东");
    },
};
let { name, age, say } = person;
console.log(name, age, say);
say();
//>>>昨夜星辰昨夜风,画楼西畔桂堂东
let {say} =person;
say()
//>>>昨夜星辰昨夜风,画楼西畔桂堂东

  1. 声明模板字符串,使用反引号"`"

    let str = `这是一个模板字符串`

      let str = `这

      let name ="张三";

//>>>我是张三

箭头函数=>

  • 声明箭头函数
    1.普通箭头函数

    let func = (a, b) => {
return a + b;
};
console.log(func(1, 2));
// >>>3

    2.简写箭头函数
    当函数只有一个参数时可以省略小括号

    let func = a => {
return a * a;
};
console.log(func(7));
// >>>3

    3.简写箭头函数
    当函数只有一条语句时可以省略大括号,此时必须省略return

    let func = a => a * a;
console.log(func(7));
// >>>3

  • 箭头函数注意事项

    1. 箭头函数的this是静态的
    2. 箭头函数不能作为对象的构造器
    3. 不能使用arguments变量
